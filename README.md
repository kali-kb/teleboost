# Teleboost Monorepo

Welcome to the Teleboost project!

This is a monorepo setup containing:
- **Frontend**: Vue.js application.
- **Backend**: NestJS application with Drizzle ORM and PostgreSQL.
- **Bot**: NestJS Telegram bot.
- **Database**: Shared database schema and utilities.

## Getting Started

Please refer to [walkthrough.md](./walkthrough.md) for detailed setup and usage instructions.

## Quick Start

```bash
pnpm install
pnpm dev
```
